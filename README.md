## Project Overview
The **Online Pet Store Management System** is a web-based application developed to support animal rescue organizations and pet shelters by providing a centralized platform to manage rescued pets, adoption processes, and medical records. The system improves transparency, accessibility, and efficiency in pet adoption management.

---

## Features
- User registration and login with role-based access (Adopter, Volunteer, Admin)
- Registration and management of rescued pets
- Adoption request submission and approval workflow
- Admin dashboard for managing users, pets, and adoption requests
- Responsive and user-friendly interface
